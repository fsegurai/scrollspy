Introduction

A lightweight, dependency-free scrollspy library that automatically highlights navigation items based on scroll position. Perfect for documentation sites, landing pages, and any content with table of contents navigation.

This library provides smooth scroll tracking and intelligent navigation highlighting with minimal setup and maximum performance.

Installation

Install the package via npm:

npm install @fsegurai/scrollspy

Or include it directly in your HTML:

<script type="module" src="path/to/scrollspy.esm.js"></script>

Basic Usage

Initialize scrollspy with minimal configuration:

import scrollspy from '@fsegurai/scrollspy';

// Initialize with navigation selector
const spy = new scrollspy('#navigation');

// Clean up when done
spy.destroy();

Configuration

Customize scrollspy behavior with various options:

Options

Available configuration options:

  • offset: Number of pixels to offset scroll position (default: 0)
  • nested: Enable nested navigation support (default: false)
  • nestedClass: CSS class for parent items (default: 'active-parent')
  • reflow: Recalculate on window resize (default: false)
  • events: Enable custom events (default: true)

Events

Listen for scrollspy events:

document.addEventListener('gumshoeactivate', (event) => {
  console.log('Active section:', event.detail.target);
});

Examples

Common usage patterns and examples:

Simple Example

Basic scrollspy setup:

const spy = new scrollspy('#nav', {
  offset: 50
});

Nested Navigation

Enable nested navigation highlighting:

const spy = new scrollspy('#nav', {
  nested: true,
  nestedClass: 'parent-active',
  offset: 100
});

Custom Offset

Adjust scroll position with custom offset:

const spy = new scrollspy('#nav', {
  offset: 120, // Account for fixed header
  reflow: true // Recalculate on resize
});

API Reference

Complete API documentation:

  • detect(): Manually trigger position detection
